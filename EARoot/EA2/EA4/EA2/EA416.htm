An object class that specifies that a date does not have to be complete. Not complete means: <br/><br/><i><u>fixing the date</u></i><br/>a date has a fixed format: 8 digits, filled in yyyymmdd (year-month-day), where only digits are allowed. <br/>when a part of the date is unknown, it is represented by zeros. The possible values of date are: <br/>yyyymmdd full date<br/>yyyymm00 day unknown<br/>yyyy000 month unknown<br/>00000000 date unknown, default value<br/><br/><i><u>sort</u></i><br/>when sorting by date, the unknown parts of a date are included in the equation. This means that, for example, a full date is always more recent than a date with an unknown day in the same month and year. So: <br/>19900505 is more recent than 19900500<br/>19900500 is more recent than 19000000<br/>19000000 is more recent than 00000000<br/>but<br/>19900500 is more recent than 19900430<br/><br/>
