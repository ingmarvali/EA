The message box is your personal mailbox for electronic messages from the government: user-friendly, fast, safe and environmentally friendly. You decide for yourself which government organizations you want to receive messages and you will no longer receive these messages by paper post. However, you will receive your messages from the tax authorities both via the message box and simply by post. By email you will receive a notification that there is a new message. <br/> <br/> the message box name of an entrepreneur is not public and may not be used for commercial purposes. <br/> only for government institutions is the message box name available. <br/> <br/>
